WHAT IS TBS?

TBS is a new method designed to compute bootstrap branch supports. The main idea is that "classical" Felsenstein Bootstrap may fail to support deep branches of large phylogenies, because the exact same branches are not found in any bootstrap tree.


Let's consider a branch b of a reference tree. This branch defines a bipartition of the taxa. It's classical bootstrap support will be computed by counting all bootstrap trees that contain the exact same bipartition of the taxa. If a branch is "almost" (but never perfectly) found in all bootstrap trees, its bootstrap support will be 0.


For this reference branch b, instead of looking for the exact same branch in bootstrap trees, TBS will search the closest branch and compute a distance with the reference branch. This distance is in computed in terms of number of taxa that must be moved around the bipartition. This allows to compute a continuous score in the interval [0,1].
